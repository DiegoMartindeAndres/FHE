# Lista de URLs para la API de Microsoft SEAL

## Compute parameters for linear regression

### GET URL

> http://localhost:3000/api/parms-linear/

### POST URLs

#### Altura vs Peso

> http://localhost:3000/api/compute-linear/?title=AlturaVsPeso&predict=175
